# César Banda — Data Analyst Portfolio

Live site: [cesar423-png.github.io](https://cesar423-png.github.io)

Personal portfolio built as a single-page site, showcasing end-to-end data analytics projects across two domains:

- **Industrial Analytics** — 4 connected projects (sales analysis, product demand relationships, export activity, demand forecasting) built on 640,000+ industrial transactions.
- **Healthcare Analytics** — SQL-based hospital resource utilization and readmission risk analysis on 89,886 patient encounters.

Each project section includes business context, methodology, an embedded interactive Power BI dashboard, key findings, and a link to the full source repository.

## Tech Stack

- HTML5 / Sass / vanilla JS
- Based on the [Hyperspace](https://html5up.net/hyperspace) template by [HTML5 UP](https://html5up.net), customized with a Power BI–inspired dark theme (`Inter` typeface, custom color palette, embedded dashboards, mobile navigation).
- Dashboards published via Power BI "Publish to Web".
- Hosted on GitHub Pages.

## Structure

```
├── index.html              # single-page site
├── assets/
│   ├── css/
│   │   ├── main.css        # compiled from Sass (theme)
│   │   └── custom.css      # portfolio-specific overrides (dashboards, mobile menu, etc.)
│   ├── sass/                # Sass source (palette in libs/_vars.scss)
│   └── js/
│       ├── main.js          # Hyperspace base behavior
│       └── mobile-menu.js   # custom mobile navigation toggle
└── images/                  # favicon, social preview
```
